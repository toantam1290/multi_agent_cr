# multi_agent_cr

DeFi Trading Agent — paper/live trading với Claude pre-mortem, opportunity screening, scalp/swing.

## Chạy nhanh

```bash
pip install -r requirements.txt
cp .env.example .env
# Điền ANTHROPIC_API_KEY, TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID vào .env
python main.py
```
